# Vue Boilerplate Template
based on vue-cli3

## Dependent plugin list
* vue2
* vue-router
* vuex
* vue-i18n
* axios
* lodash


## Project setup
```
yarn install
```

#### Compiles and hot-reloads for development
```
yarn run serve
```

#### Compiles and minifies for production
```
yarn run build
```

#### Run your tests
```
yarn run test
```

#### Lints and fixes files
```
yarn run lint
```

#### Run your unit tests
```
yarn run test:unit
```
